Network Topology

There are 4 main types of networks:

  • PAN = Personal Area Network
  • LAN = Local Area Network
  • MAN = Metropolitan Area Network
  • WAN = Wide Area Network

PAN 1m in phones, computers USB, SATA, DATA Bluetooth, infrared, microwaveBroadcast and Point-to-point
LAN 10m-1km in offices, school, houses Ethernet, coaxial cablesWi-Fi Broadcast
